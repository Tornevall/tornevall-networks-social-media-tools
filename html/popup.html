Tornevall Networks Social Media Tools

Browser companion for Tornevall Networks social media workflows, server-side AI assistance, and future admin/statistics tooling.

Requests are sent through https://tools.tornevall.net/api/ai/socialgpt/respond using your Tools bearer token.

Register at tools.tornevall.net and generate a personal bearer token there to use this extension.

More settings and Facebook admin-log tools live in Tools. For more info, open dashboard.

For custom instructions, use Tools → Social Media Tools → Facebook.
