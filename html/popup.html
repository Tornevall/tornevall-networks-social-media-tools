Tornevall Networks Social Media Tools

Browser companion for Tornevall Networks social media workflows, server-side AI assistance, fact-checking, and admin/statistics tooling.

Requests are sent through https://tools.tornevall.net/api/ai/socialgpt/respond using your Tools bearer token.

Register at tools.tornevall.net and generate a personal bearer token there to use this extension.

More settings and platform-specific tools live in Tools. For more info, open dashboard. You can also open the forum.

When this is off, Facebook admin_activities pages stay quiet and no statistics overlay is shown. When this is on, you still have to enable reporting separately on each Facebook page before anything is submitted.

Used by Verify fact. If the chosen model returns an empty/failed fact-check, the extension will retry once with gpt-4o. Dig deeper will still prefer a reasoning/thinking-capable model.

For custom instructions, use Tools → Social Media Tools → Facebook.

Changes autosave locally. Tools-backed responder settings sync automatically when a personal bearer token is present.
